# Jaydip Dungarani — Portfolio 🚀

Personal portfolio website showcasing my work as a Data Center Technician & IT Infrastructure Engineer.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **react-icons** (icon library)

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
